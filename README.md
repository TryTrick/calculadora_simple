 🧮 Calculadora Python

Una calculadora funcional con interfaz gráfica construida en Python con `tkinter`. Proyecto personal para entrenar en programacion con Python.


 ✨ Características

- **Botones redondeados** con efecto hover al pasar el cursor
- **Pantalla adaptable** — la fuente se reduce automáticamente cuando el número es largo
- **Línea de expresión** que muestra la operación realizada encima del resultado
- **Soporte de teclado físico** — escribe directamente sin usar el ratón
- **Navegación con cursor** — muévete dentro de la expresión con ← → sin borrar nada
- **Panel secundario (2nd)** — cambia los botones a funciones extra
- **Decimales periódicos** — detecta períodos y muestra el símbolo de periodicidad (0.3̄)

 🔢 Operaciones disponibles

 Panel principal
| Botón |              Función |

| `÷` `×` `+` `-`|     Operaciones básicas |
| `+/-` |              Cambiar signo del número |
| `⌫` |               Borrar último carácter |
| `AC` |               Limpiar todo |
| `.` |                Decimal |

 Panel secundario (pulsa `2nd`)
| Botón |              Función |

| `(` `)` |            Paréntesis |
| `%` |                Módulo |
| `^` |                Potencia |


 ⌨️ Atajos de teclado

| Tecla |               Acción |
|-------|--------|
| `0`–`9`, `.` |        Números y decimal |
| `+` `-` `*` `/` |     Operadores |
| `Enter` |             Calcular resultado |
| `Backspace` |         Borrar último carácter |
| `Escape` / `Delete` | Limpiar todo |
| `←` `→` |             Navegar dentro de la expresión |
